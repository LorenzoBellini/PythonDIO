Função INPUT
    É uma função built-in e é utilizzada quando queremos ler dados de entrada padrão (Teclado)
    Ela recebe um argumento do tipo string, que é exibido ao usuério na saída padrão (Tela)
    A função lê a entrada, converte pra string e retorna o valor.

-----

Função PRINT
    É uma função built-in e é utilizada quando queremos exibir dados na saída padrão (Tela)
    Recebe argumento obrigatório do tipo varargs de objetos e 4 argumentos opcionais (sep, end, file e flush)
    Todos os objetos são convertidos para string, separados por sep e terminados por end
    A string final é exibida para o usuério
